# Семинар 4. Вариант 15: Банковская система

Учебный C++-проект по теме объектно-ориентированного программирования и полиморфизма.
Предметная область проекта: банковская система со счетами разных типов, клиентами, картой, переводами и историей операций.

## Что реализовано

- абстрактный базовый класс `Account`;
- иерархия наследования:
  - `SavingsAccount`
  - `CreditAccount`
  - `DepositAccount`
- виртуальные методы и переопределение через `override`;
- полиморфная работа через базовый тип `Account`;
- контейнер `std::vector<std::unique_ptr<Account>>`;
- хранение объектов разных типов в одном контейнере;
- вызов виртуальных методов для разных наследников;
- пример `dynamic_cast`;
- сохранена рабочая сборка проекта через CMake.

## Основные классы

- `Bank` - банк и операции перевода;
- `Department` - отделение банка;
- `Client` - клиент банка;
- `Account` - абстрактный базовый класс счета;
- `SavingsAccount` - сберегательный счет;
- `CreditAccount` - кредитный счет;
- `DepositAccount` - депозитный счет;
- `Card` - банковская карта;
- `Transaction` - операция по счету;
- `CommissionCalculator` - расчет комиссии.

## Что демонстрирует программа

После запуска программа:

- создает банк, отделения и клиентов;
- открывает счета разных типов;
- выполняет пополнение, снятие, оплату картой и перевод;
- выводит итоговое состояние счетов;
- показывает историю операций;
- демонстрирует полиморфизм через `std::vector<std::unique_ptr<Account>>`;
- показывает пример проверки типа через `dynamic_cast`.

## Сборка и запуск
```bash
cmake -B build && cmake --build build && ./build/project_bank
```
